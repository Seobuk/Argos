# 행동 강령 / Code of Conduct

Argos 커뮤니티(이슈, PR, 토론)는 누구에게나 열려 있고 서로 존중하는 공간이어야 합니다.
The Argos community (issues, PRs, discussions) is open to everyone and built on mutual respect.

## 기대하는 행동 / Expected behavior

- 서로 다른 관점과 경험 수준을 존중합니다.
- 건설적인 피드백을 주고, 정중하게 받습니다.
- 기술적 의견 차이는 코드와 근거로 이야기합니다.

## 용납되지 않는 행동 / Unacceptable behavior

- 인신공격, 모욕, 차별적 발언, 괴롭힘
- 타인의 개인정보 무단 공개
- 그 밖에 합리적인 사람이 부적절하다고 여길 행동

## 적용 및 신고 / Enforcement & reporting

이 강령은 이 저장소의 모든 공간(이슈, PR, 코드 리뷰, 토론)에 적용됩니다.
위반 사례는 저장소 관리자에게 [비공개 보안 권고](https://github.com/Seobuk/Argos/security/advisories/new)
또는 GitHub 프로필의 연락처로 알려주세요. 관리자는 부적절한 댓글·커밋·이슈를 수정하거나
삭제할 수 있으며, 반복 위반 시 참여를 제한할 수 있습니다.

이 문서는 [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 2.1을
바탕으로 한 요약본입니다. This is a condensed adaptation of the Contributor Covenant v2.1.
